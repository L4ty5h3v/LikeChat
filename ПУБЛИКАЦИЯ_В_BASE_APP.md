# Как опубликовать приложение в Base App

## Текущий статус

Ваше приложение уже настроено для Base App:
- **App ID:** `693c50258a7c4e55fec73fe1`
- **URL приложения:** `https://likechat-base-app.vercel.app`
- **Мета-тег:** `<meta name="base:app_id" content="693c50258a7c4e55fec73fe1" />` (уже добавлен в `pages/_document.tsx`)

## Шаги для публикации

### 1. Войдите в Base App Developer Portal

1. Откройте https://base.org или https://base.app
2. Войдите через кошелек Base (тот же, который используется в приложении)
3. Перейдите в раздел **Developer Portal** или **My Apps**

### 2. Найдите или создайте приложение

Если приложение уже создано (по App ID `693c50258a7c4e55fec73fe1`):
- Откройте его в панели управления
- Проверьте настройки

Если приложение еще не создано:
- Нажмите **"Create New App"** или **"Add App"**
- Заполните форму:
  - **App Name:** LikeChat Base (или другое название)
  - **App URL:** `https://likechat-base-app.vercel.app`
  - **Description:** Support posts on Base - buy post tokens for $0.10 USDC
  - **Icon:** Загрузите иконку приложения (рекомендуется 512x512px)

### 3. Настройте метаданные приложения

В Base App Developer Portal укажите:

- **App Name:** LikeChat Base
- **Short Description:** Buy post tokens to support creators on Base
- **Long Description:** 
  ```
  LikeChat Base - взаимная поддержка постов на Base через покупку post-token.
  
  - Покупайте посты за $0.10 USDC
  - После 5 покупок публикуйте свой пост
  - Поддерживайте создателей контента на Base
  ```
- **Category:** Social / Community
- **Icon:** Загрузите иконку (PNG, 512x512px, прозрачный фон)
- **Screenshot:** Загрузите скриншоты приложения (рекомендуется 3-5 штук)

### 4. Проверьте настройки безопасности

Убедитесь, что:
- ✅ Приложение доступно по HTTPS (Vercel автоматически предоставляет)
- ✅ CORS настроен правильно (если нужен)
- ✅ Все API endpoints работают корректно

### 5. Отправьте на модерацию

1. В панели управления приложением нажмите **"Submit for Review"** или **"Publish"**
2. Дождитесь проверки (обычно 1-3 рабочих дня)
3. После одобрения приложение появится в каталоге Base App

## Проверка перед публикацией

### ✅ Убедитесь, что:

1. **Приложение работает:**
   - Откройте https://likechat-base-app.vercel.app
   - Проверьте все страницы (`/`, `/tasks`, `/submit`)
   - Проверьте покупку поста (тестовая транзакция)

2. **Мета-тег присутствует:**
   - Откройте https://likechat-base-app.vercel.app
   - Откройте DevTools (F12) → Elements
   - Найдите `<meta name="base:app_id" content="693c50258a7c4e55fec73fe1" />`

3. **Все функции работают:**
   - Авторизация через Base кошелек
   - Покупка постов ($0.10 USDC)
   - Публикация новых постов
   - Открытие постов в новой вкладке

## После публикации

1. **Проверьте в Base App:**
   - Откройте Base App на мобильном устройстве
   - Найдите ваше приложение в каталоге
   - Откройте его и протестируйте

2. **Мониторинг:**
   - Следите за логами в Vercel Dashboard
   - Проверяйте ошибки в консоли браузера
   - Отслеживайте использование через аналитику (если настроена)

## Если возникли проблемы

1. **Проверьте App ID:**
   - Убедитесь, что `base:app_id` в `pages/_document.tsx` совпадает с ID в Base App Developer Portal

2. **Проверьте URL:**
   - Убедитесь, что URL в Base App Developer Portal совпадает с `https://likechat-base-app.vercel.app`

3. **Проверьте мета-теги:**
   - Откройте исходный код страницы (View Source)
   - Убедитесь, что `<meta name="base:app_id">` присутствует

4. **Свяжитесь с поддержкой Base:**
   - Если приложение не появляется в каталоге после модерации
   - Если есть проблемы с интеграцией
